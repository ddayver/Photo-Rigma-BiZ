)

Функция проверки содержимого на соотвествие регулярному выражению

Аргументы

$regexpуказывает, проверять ли функции, что $field соответствует регулярному выражению (по-умолчанию False - не проверять)
$not_zeroуказывает, проверять ли функции, что $field не равно нулю (по-умолчанию False - не проверять)
Возвращает
False, если $field не соотвествует заданному набору параметров, иначе True.

work::check_get, work::check_post, work::check_session

$not_zeroуказывает, проверять ли функции, что $_GET[$field] не равно нулю (по-умолчанию False - не проверять)
Возвращает
