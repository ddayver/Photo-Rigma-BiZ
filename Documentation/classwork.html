Подробное описание

Общий класс (набор функций) + хранилище для данных о конфигурации.

Данный класс содержит набор общих для всей игры функций, а так же используется для хранения данных о конфигурации.

Функция формирует информационную строку по конктретному разделу

Аргументы:
$cat_idсодержит идентификатор раздела или, если $user_flag = 1,то идентификатор пользователя
$user_flagфлаг, указывающий формировать ли обычный список разделов (0) или список пользовательских альбомов (1)

$  regexp = false,
$  not_zero = false 
)

Функция проверки содержимого на соотвествие регулярному выражению

Аргументы:
$fieldсодержит значение, которое надо проверить (обязательное поле)
$regexpуказывает, проверять ли функции, что $field соответствует регулярному выражению (по-умолчанию False - не проверять)
$not_zeroуказывает, проверять ли функции, что $field не равно нулю (по-умолчанию False - не проверять)
Возвращает:
False, если $field не соотвествует заданному набору параметров, иначе True.
См. также:
work::check_get, work::check_post, work::check_session

Функция проверки данных, переданных через $_GET[].

$emptyуказывает, проверять ли функции, что $_GET[$field] не является пустым (по-умолчанию False - не проверять)
$regexpуказывает, проверять ли функции, что $_GET[$field] соответствует регулярному выражению (по-умолчанию False - не проверять)
$not_zeroуказывает, проверять ли функции, что $_GET[$field] не равно нулю (по-умолчанию False - не проверять)
Возвращает:
False, если $_GET[$field] не соотвествует заданному набору параметров, иначе True.
