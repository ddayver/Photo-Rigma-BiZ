Photo Rigma BiZ

Photo Rigma BiZ — это веб-приложение для создания галереи изображений с поддержкой пользовательских альбомов, рейтингов, комментариев и администрирования. Проект разработан с использованием PHP, MySQL и HTML/CSS, что делает его легким для развертывания и использования.

Описание проекта

Photo Rigma BiZ предоставляет удобный интерфейс для загрузки, просмотра и управления изображениями. Пользователи могут создавать свои альбомы, оценивать фотографии, оставлять комментарии и взаимодействовать с другими участниками. Администраторы имеют расширенные права для управления контентом, пользователями и настройками платформы.

Основная целевая аудитория - учебные заведения. Позволяет выставлять отметки фотографиям как самим участникам, так и "модераторам" (например пед.состав).

Основные функции

  • Загрузка изображений: Пользователи могут загружать изображения в свои альбомы.
  • Пользовательские альбомы: Каждый пользователь может создавать и управлять своими альбомами.
  • Рейтинги и оценки: Возможность оценивать изображения и просматривать топовые фотографии.
  • Администрирование: Администраторы могут управлять пользователями, изображениями и настройками платформы.
  • Защита от SQL-инъекций и XSS: Встроенные механизмы защиты от вредоносных запросов.
  • Мультиязычность: Поддержка нескольких языков через файлы локализации.
  • Генерация эскизов: Автоматическое создание миниатюр для загруженных изображений.

Требования к системе

Для работы приложения необходимы следующие компоненты:

  • Веб-сервер: Apache или Nginx
  • PHP: Версия 8.0 или выше
  • MySQL: Версия 5.7 или выше
  • Расширения PHP:
    • PDO
    • GD (для обработки изображений)
    • JSON
    • MBString
  • Права доступа: Необходимы права на запись в директории для загрузки изображений и логов.

Установка

  1. Склонируйте репозиторий:
    git clone https://github.com/ddayver/Photo-Rigma-BiZ.git
    cd Photo-Rigma-BiZ
  2. Настройте базу данных:
    • Создайте новую базу данных MySQL.
    • Импортируйте структуру базы данных из файла database.sql (если он существует).
      mysql -u username -p database_name < database.sql
  3. Настройте конфигурацию:
    • Откройте файл config.php и укажите параметры подключения к базе данных:
      define('DB_HOST', 'localhost');
      define('DB_NAME', 'your_database_name');
      define('DB_USER', 'your_database_user');
      define('DB_PASS', 'your_database_password');
  4. Настройте права доступа:
    • Убедитесь, что веб-сервер имеет права на запись в следующие директории:
      • avatar/
      • gallery/
      • logs/
      • thumbnail/
  5. Запустите приложение:
    • Откройте браузер и перейдите по адресу вашего сайта (например, http://localhost/Photo-Rigma-BiZ).

Использование

Регистрация и вход

  • Перейдите на страницу регистрации и создайте учетную запись.
  • После регистрации войдите в систему на странице входа.

Загрузка изображений

  • Перейдите в свой профиль и создайте новый альбом.
  • Загрузите изображения в альбом через форму загрузки.

Оценка и комментирование

  • Просмотрите изображения других пользователей.
  • Оцените изображение или оставьте комментарий.

Администрирование

  • Войдите в систему как администратор.
  • Управляйте пользователями, изображениями и настройками платформы через панель администратора.
