\\PhotoRigma\\Classes\\Database::validate_array

Рекурсивная проверка массива на корректность ключей и значений.

Рекурсивная проверка массива на корректность ключей и значений. Метод выполняет рекурсивную проверку массива на соответствие следующим условиям:

  1. Все ключи должны быть строками.
  2. Все значения должны быть либо строками, либо вложенными массивами.
  3. Глубина массива не должна превышать указанное значение $max_depth.

Этот метод является приватным и предназначен только для использования внутри класса.

Предупреждения
Метод чувствителен к глубине массива. Убедитесь, что параметр $max_depth установлен корректно.
Не используйте метод для очень больших массивов из-за риска переполнения стека.
См. также
\PhotoRigma\Classes\Work::set_lang Сеттер для установки массива $lang.
Аргументы
array$arrayМассив для проверки. Пустые массивы недопустимы.
int$max_depthМаксимальная допустимая глубина массива (должна быть положительным целым числом).
int$current_depthТекущая глубина рекурсии (используется внутренне для контроля максимальной глубины).
Возвращает
array Массив ошибок. Каждый элемент представляет собой строку с описанием проблемы. Пример: ['Ключ должен быть строкой', 'Глубина превышена'].
$data = [
'key1' => 'value1',
'key2' => [
'nested_key' => 'nested_value',
],
];
$errors = $this->validate_array($data, 2);
if (!empty($errors)) {
echo "Обнаружены ошибки:\n";
foreach ($errors as $error) {
echo "- {$error}\n";
}
} else {
echo "Массив прошёл проверку успешно.";
}
