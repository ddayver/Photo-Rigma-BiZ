Конструктор класса, инициализирующий основные параметры сайта и темы оформления.

Метод выполняет проверку входных данных, инициализирует свойства класса и вычисляет пути к директориям тем. Выполняется автоматически при создании объекта класса.

Аргументы
string$site_urlURL сайта. Должен быть валидным URL.
string$site_dirДиректория сайта. Должна существовать и быть доступной для чтения.
string$themeИмя темы оформления. Должно содержать только латинские буквы, цифры, дефисы и подчеркивания.
bool$mod_rewriteВключение читаемых URL. По умолчанию false.
Исключения
InvalidArgumentExceptionЕсли $site_url не является валидным URL.
InvalidArgumentExceptionЕсли $site_dir не существует или не является директорией.
InvalidArgumentExceptionЕсли $theme пустое или содержит недопустимые символы.
RuntimeExceptionЕсли директория тем не найдена.
RuntimeExceptionЕсли нет прав доступа к директории тем.

Пример создания объекта:

'https://example.com',
'/var/www/example',
'default',
true
);

◆ add_case()

add_case ( string $name,
string $value,
string|false $path_array = false )

Метод добавляет данные о выборе блока для вывода фрагментов шаблона, с возможностью рекурсивного размещения.

Метод выполняет следующие шаги:

  1. Проверяет корректность имени условия (только латинские буквы, цифры и подчеркивания).
  2. Проверяет корректность значения условия (только латинские буквы, цифры и подчеркивания).
  3. Проверяет формат пути ($path_array), если он указан.
  4. Если путь не указан, добавляет условие в массив block_case. Если путь указан, рекурсивно добавляет условие в объекты. Метод является публичным и предназначен для прямого использования извне.
См. также
PhotoRigma\Classes\Template\$block_case Свойство, содержащее массив данных о выборе блока.
PhotoRigma\Classes\Template\$block_object Свойство, содержащее объекты для рекурсивного размещения условий.
PhotoRigma\Classes\Template\test_is_object() Метод, используемый для разбора пути.
Аргументы
string$nameНазвание условия. Должно содержать только латинские буквы, цифры и подчеркивания.
string$valueЗначение условия. Должно содержать только латинские буквы, цифры и подчеркивания.
string | false$path_arrayПуть для рекурсивного размещения условия. Должен быть строкой (в виде: Массив1[0]->Массив1.0[0]) или false (по умолчанию).
Исключения
InvalidArgumentExceptionЕсли имя условия некорректно (пустое или содержит недопустимые символы).
InvalidArgumentExceptionЕсли значение условия некорректно (пустое или содержит недопустимые символы).
InvalidArgumentExceptionЕсли путь ($path_array) имеет некорректный формат (не строка и не false).
RuntimeExceptionЕсли результат метода test_is_object() некорректен.
InvalidArgumentExceptionЕсли формат next_path некорректен (не строка и не false).
Предупреждения
Метод особенно чувствителен к параметру $path_array. Убедитесь, что он передается в правильном формате (строка или false).

Пример вызова метода:

$object = new \PhotoRigma\Classes\Template('https://example.com', '/var/www/example', 'default');
$object->add_case('BLOCK_TYPE', 'HEADER');
$object->add_case('BLOCK_TYPE', 'FOOTER', 'Массив1[0]->Массив1.0[0]');

  4. Добавляет каждую пару ключ-значение в шаблон с помощью метода add_case(). Метод является публичным и предназначен для прямого использования извне.

string | false$path_arrayПуть для рекурсивного размещения условия. Должен быть строкой (в виде: Массив1[0]->Массив1.0[0]) или false (по умолчанию).

  2. Формирует ключи для поиска блоков в шаблоне: <!-- ARRAY_НАЗВАНИЕ_BEGIN --> и <!-- ARRAY_НАЗВАНИЕ_END -->.

bool $txt = false )
private

Метод преобразует URL в более читаемый вид с использованием правил mod_rewrite, если они включены.

Метод выполняет следующие шаги:

  1. Проверяет, включен ли режим mod_rewrite (свойство $this->mod_rewrite).
  2. В зависимости от флага $txt определяет формат окончания ссылки.
  3. Определяет шаблоны для замены URL на основе параметров запроса (например, ?action=profile&id=123 → profile/id_123.html).
  4. Выполняет замену URL в переданном содержимом ($content) с использованием регулярных выражений.
  5. Возвращает обработанное содержимое. Метод является приватным и предназначен только для использования внутри класса.
