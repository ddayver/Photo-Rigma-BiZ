\\PhotoRigma\\Classes\\Work::set_lang

Установка массива языковых данных через сеттер.

Установка массива языковых данных через сеттер. Этот метод позволяет установить массив языковых данных ($lang). Метод выполняет следующие действия:

  1. Проверяет, что массив не пустой.
  2. Рекурсивно проверяет корректность ключей и значений с помощью метода validate_array.
  3. Обрабатывает массив через метод sanitize_array, который использует Work::clean_field.
  4. Логирует изменения, если они были выполнены.
  5. Передаёт обработанные данные в дочерние классы (Work_Template и Work_CoreLogic) через их методы set_lang.
См. также
PhotoRigma\Classes\Work\validate_array Рекурсивная проверка массива на корректность ключей и значений.
PhotoRigma\Classes\Work\sanitize_array Рекурсивная обработка массива через Work::clean_field.
PhotoRigma\Classes\Work_CoreLogic\$lang Свойство дочернего класса Work_CoreLogic.
PhotoRigma\Classes\Work_Template\$lang Свойство дочернего класса Work_Template.
PhotoRigma\Include\log_in_file() Логирует ошибки.
Аргументы
array$langМассив языковых данных:
  • Не должен быть пустым.
  • Ключи и значения должны соответствовать ограничениям (глубина до 4 уровней).
Исключения
InvalidArgumentExceptionЕсли массив некорректен (пустой или содержит некорректные ключи/значения).
Заметки
Метод рекурсивно проверяет и обрабатывает массив языковых данных. Изменения логируются только при наличии изменений.
Предупреждения
Некорректные данные (пустой массив или некорректные ключи/значения) вызывают исключение. Массив должен соответствовать ограничению глубины до 4 уровней.
Необходимо сделать
Интегрировать в систему кеширования языковых переменных.
// Пример использования метода
$work = new \PhotoRigma\Classes\Work();
$lang = [
'key1' => 'value1',
'key2' => [
'subkey1' => 'subvalue1'
]
];
$work->set_lang($lang);
