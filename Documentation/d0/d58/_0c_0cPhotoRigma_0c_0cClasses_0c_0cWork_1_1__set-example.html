\\PhotoRigma\\Classes\\Work::__set

Устанавливает значение свойства $config.

Устанавливает значение свойства $config. Этот метод вызывается автоматически при попытке установить значение недоступного свойства. Доступ разрешён только к свойству $config. Если запрашивается другое свойство, выбрасывается исключение Exception. Значение $config должно быть массивом, где ключи и значения являются строками. При успешном обновлении конфигурации:

  • Изменения логируются с помощью функции log_in_file (за исключением ключей из списка exclude_from_logging).
  • Обновлённая конфигурация передаётся в дочерние классы ($this->image, $this->template, $this->core_logic) через их свойства config.
См. также

PhotoRigma\Classes\Work_Image\$config Свойство дочернего класса Work_Image.

Аргументы
string$nameИмя свойства:
  • Допустимое значение: 'config'.
  • Если указано другое имя, выбрасывается исключение.
mixed$valueЗначение свойства:
  • Должен быть массивом, где ключи и значения являются строками.
Исключения
InvalidArgumentExceptionЕсли значение некорректно (не массив или содержатся некорректные ключи/значения).
ExceptionЕсли запрашивается несуществующее свойство.
Заметки
Этот метод предназначен только для изменения свойства $config. Логирование изменений выполняется только для определённых ключей.
Предупреждения
Некорректные данные (не массив, нестроковые ключи или значения) вызывают исключение. Попытка установки значения для несуществующего свойства также вызывает исключение.
// Пример использования метода
$work = new \PhotoRigma\Classes\Work();
$work->config = [
'theme' => 'dark',
'language' => 'en'
];
