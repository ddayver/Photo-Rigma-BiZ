\\PhotoRigma\\Classes\\Database::sanitize_array

Рекурсивная обработка массива для безопасного вывода через HTML.

Рекурсивная обработка массива для безопасного вывода через HTML. Метод выполняет рекурсивную обработку массива, вызывая Work::clean_field для каждого строкового значения. Если значение изменяется в процессе очистки, фиксируются изменения в массиве $changes. Исходный массив модифицируется по ссылке.

Этот метод является приватным и предназначен только для использования внутри класса.

См. также
\PhotoRigma\Classes\Work::set_lang Сеттер для установки массива $lang.
\PhotoRigma\Classes\Work::clean_field() Обработка значений для безопасного вывода через HTML.
Аргументы
array$arrayМассив для обработки. Пустые массивы недопустимы.
Возвращает
array Массив изменений. Каждый элемент представляет собой ассоциативный массив с ключами:
  • 'original' (string): Исходное значение.
  • 'cleaned' (string): Очищенное значение. Пример: ['key' => ['original' => 'unsafe', 'cleaned' => 'safe']].
Предупреждения
Метод изменяет исходный массив по ссылке. Убедитесь, что это допустимо в контексте использования.
Не используйте метод для очень больших массивов из-за риска переполнения стека.
Необходимо сделать
Планируется интеграция в систему кеширования языковых параметров.
$data = [
'title' => '<script>alert("XSS")</script>',
'description' => 'Безопасное описание',
'nested' => [
'unsafe_key' => '<img src=x onerror=alert(1)>',
],
];
$changes = $this->sanitize_array($data);
if (!empty($changes)) {
echo "Обнаружены изменения:\n";
foreach ($changes as $key => $change) {
echo "- Ключ '{$key}':\n";
echo " Исходное значение: {$change['original']}\n";
echo " Очищенное значение: {$change['cleaned']}\n";
}
} else {
echo "Массив не требует очистки.";
}
