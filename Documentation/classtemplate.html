Подробное описание

Новый класс работы с шаблонами.

Данный класс содержит функции по работе с шаблонами (наполнение данными, обработка). Внедряется взамен старого.

См. также:
template_old

Добавление данных о выборе блока для вывода фрагментов шаблона

Добавляет данные о выборе блока для вывода фрагментов шаблона как в основе шаблона, так и в рекурсивных массивах

Аргументы:
$nameсодержит название пемеренной условия
$valueсодержит значение переменной (наименование или номер блока)
$path_arrayсодержит путь, по которому рекурсивно необходимо разместить переменную в виде: Массив1[0]->Массив1.0[0] (по-умолчанию False)
См. также:
$block_case, $block_object, test_is_object
