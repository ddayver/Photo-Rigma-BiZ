Подробное описание

Класс работы с шаблонами.

Данный класс содержит функции по работе с шаблонами (наполнение данными, обработка).

Методы

add_if ( $  name,
$  value,
$  path_array = false 
)

Добавление данных об условиях вывода фрагментов шаблона

Добавляет данные об условиях вывода фрагментов шаблона как в основе шаблона, так и в рекурсивных массивах

Аргументы:
$nameсодержит название пемеренной условия
$valueсодержит значение переменной (True или False)
$path_arrayсодержит путь, по которому рекурсивно необходимо разместить переменную в виде: Массив1[0]->Массив1.0[0] (по-умолчанию False)
См. также:
$block_if, $block_object, test_is_object
