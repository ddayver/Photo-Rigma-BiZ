Подробное описание

Класс для обеспечения безопасности приложения.

Этот класс предоставляет методы для проверки входных данных, защиты от спам-ботов, фильтрации email-адресов и других задач, связанных с безопасностью приложения.

См. также
\PhotoRigma\Classes\Work Все методы этого класса вызываются через класс Work.
Заметки
Этот класс реализует интерфейс Work_Security_Interface.
Предупреждения
Не используйте этот класс напрямую, все вызовы должны выполняться через класс Work.
Примеры
/home/victoria/Photo-Rigma-BiZ/www/include/work.php.

Внутренний метод для проверки содержимого поля на соответствие регулярному выражению или другим условиям.

Метод выполняет следующие проверки:

  • Если задано регулярное выражение ($regexp), проверяет соответствие значения этому выражению.
  • Если флаг $not_zero установлен, проверяет, что значение не является числом 0.
  • Проверяет, что значение не содержит запрещённых паттернов из $this->compiled_rules.
См. также
\PhotoRigma\Classes\Work_Security::check_field() Публичный метод, который вызывает этот внутренний метод.
Аргументы
string$fieldЗначение поля для проверки.
string | false$regexpРегулярное выражение (необязательно). Если задано, значение должно соответствовать этому выражению.
bool$not_zeroФлаг, указывающий, что значение не должно быть числом 0.
Возвращает
bool True, если поле прошло все проверки, иначе False.

Внутренний метод для замены символов в email-адресах.

Метод заменяет символы '@' и '.' на '[at]' и '[dot]', чтобы затруднить автоматический парсинг email-адресов ботами. Проверяет, что входной email не пустой и соответствует формату email.
