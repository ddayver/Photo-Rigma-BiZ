Устаревший класс работы с шаблонами.

Данный класс содержит устаревшие функции по работе с шаблонами (наполнение данными, обработка). Будет заменен на новый.

template

  $id_photo = 0 
)

Функция генерирует блок вывода последнего, лучшего, случайного или указанного изображения

Аргументы:
$typeесли значение равно 'top' - вывести лучшее фото по оценкам пользователя, если 'last' - последнее добавленое фото, если 'cat' - вывести фото, указанное в $id_photo, если не равно пустому - вывести случайное изображение
$id_photoесли $type равно 'cat' - выводит фото с указанным идентификатором
Возвращает:
Сформированный HTML-код блока изображения

db, work, user, create_template
